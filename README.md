# Bulk Content Management

Bulk Content Management is a WordPress admin plugin for generating, importing, exporting, and assigning posts and taxonomy terms in bulk.

It supports local prompt-assisted generation without an external AI API, AJAX batching for large jobs, post content templates, taxonomy-aware term generation, post-to-term assignment, and focused CSV workflows for posts and terms.

## Current Capabilities

- Generate flat or nested taxonomy terms.
- Assign generated terms to selected posts.
- Generate posts with configurable status, post type, content, and term assignment.
- Run large term and post generation jobs through AJAX batches with pause, resume, and stop controls.
- Import and export posts and terms through separated Import / Export tabs.
- Create same-name terms with different slugs from CSV.
